Bar Chart Project

Single Valued Horizontal Bar Chart

Do you need to quickly generate a bar chart to show data? This simple easy to use API will allow you to generate a horizontal bar chart or a stacked horizontal bar chart quickly and easily on you page. The graph below shows shows a single bar chart with bar labels and the numeric value added to each bar.

Multivalued Horizontal Bar Chart

In addition to single value bar chart, you can easily add miltivalued bar charts. You can have stack colors when refernced in your tables, or create random colored bars on your chart.
